# minimalist-task-tracker

A simple todo app where users can add, edit, and delete tasks. The app features a clean and minimalist design with intuitive controls for task management. Users can also mark tasks as complete or incomplete and filter tasks based on their status.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/minimalist-task-tracker.git
cd minimalist-task-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
